**Tasks (বাংলা অনুবাদ)**
React প্রোজেক্টকে স্কেল-রেডি রাখতে নিচের ডেলিভারেবলগুলো নিশ্চিত করতে হবে:

1. **Vite + TailwindCSS দিয়ে নতুন React প্রোজেক্ট তৈরি করো** অথবা তোমার বিদ্যমান প্রোজেক্ট ব্যবহার করো।
2. **একটি বা একাধিক কম্পোনেন্ট বানাও** যেগুলো একটি user array (৫ জন user) থেকে ডাটা নিয়ে UI-তে full name, age, occupation দেখাবে।
3. **প্রতিটি user-এর জন্য unique key ব্যবহার করতে হবে**, ঠিক যেভাবে lesson-এ শেখানো হয়েছে।
4. **Fragment ব্যবহার করতে হবে** (`<> </>` বা `<React.Fragment>`).
5. পেজের **ডানদিকে টপ-রাইট কর্নারে একটি “+” আইকন থাকবে**।
   ক্লিক করলেই Math.random() ব্যবহার করে **নতুন random user UI-তে যোগ হবে**।
   নতুন user নেওয়ার জন্য কোনো input UI তৈরি করা লাগবে না।
6. উপরে কোথাও **Total user count দেখাতে হবে**, এবং প্রতিবার নতুন user যোগ হলে এটি রিয়েল-টাইমে আপডেট হবে।
7. **প্রথমে UI-র একটি স্কেচ তৈরি করতে হবে** – Figma, Excalidraw, draw.io, বা MS Paint যেকোনো কিছুতে।
8. **Component Breakdown Diagram তৈরি করো**
9. **Component Hierarchy Diagram তৈরি করো**
10. **State এবং Props চিহ্নিত করো**
11. **অ্যাপটি পাবলিকলি ডিপ্লয় করো**
12. **README ফাইল তৈরি করো**, যেখানে থাকবে:

    - তোমার ডিজাইন করা UI
    - সব Diagram
    - কীভাবে প্রোজেক্ট লোকালি রান করতে হবে
    - Deploy URL
